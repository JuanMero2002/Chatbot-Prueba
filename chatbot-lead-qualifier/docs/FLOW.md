# Flujo de la Aplicación

## Flujo del Chatbot

1. Usuario inicia conversación
2. Chatbot recibe mensaje
3. Intent Classifier identifica la intención
4. Response Generator prepara respuesta
5. Lead Qualifier evalúa al usuario
6. Se almacena información del lead
7. Se envía respuesta al usuario

## Flujo de Calificación de Leads

1. Sistema recopila información BANT
   - Budget (Presupuesto)
   - Authority (Autoridad)
   - Need (Necesidad)
   - Timeline (Cronograma)

2. Lead Scorer calcula puntuación
3. Lead Classifier determina categoría
4. Se registra estado de calificación
5. Se notifica a equipo de ventas
